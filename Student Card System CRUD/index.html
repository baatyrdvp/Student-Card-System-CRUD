Student Card System

Для чего нужна данная система?

Ни для кого не секрет, что сейчас в 21 веке идет активная цифровизация абсолютного всего, начиная обычными документами,
и заканчивая целыми стартапами и бизнесами. Данная система нужна для того, чтобы идентифицировать каждого студента в системе
университета. Благодаря этой системе, каждый студент сможет отслеживать свои успехи в учебе, он сможет видеть какие баллы он получил
и сколько раз он пропускал занятия.
